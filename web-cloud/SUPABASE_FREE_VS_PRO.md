# Supabase Free vs Pro Plan Comparison

## Overview

This document compares the Supabase Free and Pro plans, highlighting key features, pricing, and suitability for different project needs.

## Free Plan

### Price
$0/month

### Best For
- MVPs, personal projects, prototyping, learning

### Key Features and Limits
- Up to 2 projects
- 500 MB database storage
- 1 GB file storage
- Up to 50,000 MAU
- Unlimited API requests
- Shared CPU, 500 MB RAM
- 5 GB egress, 5 GB cached egress
- Community support
- Projects paused after inactivity
- Limited backups and observability
- Basic Auth & Realtime
- Limited Edge Functions

### Limitations
- Limited resources, not for production
- No point-in-time recovery or advanced features

## Pro Plan

### Price
Starting at $25/month (includes $10 compute credits). Usage-based pricing applies.

### Best For
- Production apps, growing businesses

### Included Resources
- Up to 100,000 MAU
- 8 GB database
- 100 GB file storage
- 250 GB egress
- 250 GB cached egress
- Dedicated compute options
- Daily backups
- Point-in-time recovery
- 7-day log retention
- Email + Community support
- Usage-based scaling
- 2M Edge Function invocations

### Benefits
- More resources, dedicated support, production-ready

## Which Plan is Right for You?

### Choose Free If:
- Testing, prototyping, small personal projects.

### Choose Pro If:
- Launching a production app with user growth.

## Important Considerations
- Monitor usage to avoid overage costs. Pro is essential for real-world applications.
